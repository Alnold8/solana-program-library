# Extensions Documentation

This document outlines three planned extensions for the Solana Program Library:

## 1. Dynamic Fee Tiers for Token Program
### Specification
- Implement a system of dynamic fee tiers that adjust based on transaction volume and market conditions. This aims to incentivize higher transaction throughput and provide better pricing for users.

### Implementation Roadmap
- **Q2 2026**: Research & Design Phase
- **Q3 2026**: Prototype Development
- **Q4 2026**: Testing & Validation

## 2. Weighted Voting by Tier for Governance Program
### Specification
- Introduce a weighted voting mechanism where token holders can have their votes weighted based on their staked quantity or tier of tokens held. This increases the influence of users who commit more resources to the governance.

### Implementation Roadmap
- **Q2 2026**: Requirements Gathering
- **Q3 2026**: Development of Smart Contracts
- **Q4 2026**: Community Review & Deployment

## 3. Limit Orders for Token Swap Program
### Specification
- Enable users to set limit orders for token swaps, allowing them to execute trades only when specific price conditions are met, enhancing the trading flexibility.

### Implementation Roadmap
- **Q2 2026**: Design Specification & Feedback
- **Q3 2026**: Development Phase
- **Q4 2026**: Integration & Testing
